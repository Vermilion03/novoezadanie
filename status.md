## git status


**git status** - показывает состояние файлов в рабочей дириктории и индексе:
какие файлы изменены, но не добавлены в индекс, какие ожидают коммита в индексе.
__Вдобавок к этому выводятся подсказки о том , как изменить состояние файлов.__


```bash=
git status
```
